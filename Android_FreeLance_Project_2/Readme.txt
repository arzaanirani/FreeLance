This app uses Ionic Framework which uses Cordova to build hybrid mobile apps. 

You need to have ionic platform installed to develop and build this app. An already build version of the app is present in published/android/ folder

To install ionic framework run 'npm install -g cordova ionic' inside the app folder

To build the app, you need to run the following commands

1) ionic platform add android
2) sh publish.sh
3) Enter Passphrase for keystore as 'assignment3' without quotes
4) The app apk will be created inside published/Android folder
